# Chrome Pretty JSON

Automatically formats and syntax-highlights JSON responses in Chrome.

## Features

- Detects JSON pages and formats them with syntax highlighting
- Six built-in color themes: Gruvbox Dark, Catppuccin Dark, GitHub, VS Code Dark, Monokai, Dracula
- Theme preference saved across sessions
- Runs entirely locally with no external requests

## Install

1. Clone this repo
2. Open `chrome://extensions`
3. Enable Developer Mode
4. Click "Load unpacked" and select this directory

## Usage

Navigate to any URL that returns JSON. The extension detects it automatically and renders it with line numbers and syntax highlighting. Click the extension icon to switch themes.
